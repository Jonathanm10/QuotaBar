# Code Of Conduct

## Our Standard

Participation in the QuotaBar project should be respectful, direct, and constructive.

Examples of expected behavior:
- giving technical feedback tied to the code or behavior
- assuming good faith while still being precise about problems
- helping keep issues and pull requests focused and reviewable
- respecting other contributors' time and boundaries

Examples of unacceptable behavior:
- harassment, insults, or intimidation
- discriminatory or demeaning language
- publishing private information, credentials, or logs that expose another person
- derailing technical discussions into personal conflict
- repeated bad-faith interaction after being asked to stop

## Scope

This Code of Conduct applies in project spaces including issues, pull requests, code review, and other repo-linked discussions.

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, pull requests, and other contributions that violate this Code of Conduct.

For urgent or sensitive conduct concerns, do not post private details publicly. Request a private contact path through the repository maintainers instead.
